Add a machine

On any Mac or Linux box you want to drive from here, with tmux running, sign in once with the same Google account you log into the site with:

git clone https://github.com/cjzeroxaa/tmux-mobile && cd tmux-mobile
npm ci
node server.mjs --register https://eng.impo.ai --login

The CLI prints a Google device-login code. After approval, the machine appears in everyone's picker who shares your workspace (see below). Full instructions: docs/join-network.md.

Who can see what

Sign-in is open to any Google account, but what you see depends on the account type:

  • Google Workspace users (e.g. @rebyte.ai) see every machine registered by anyone in the same workspace.
  • Personal Google accounts (e.g. @gmail.com) only see their own machines.
  • Super-admins (deployment config, SUPER_ADMIN_EMAILS) see every machine on the controller regardless of workspace.

Each agent card shows who registered it. To make a machine invisible to your workspace, stop its agent process.
