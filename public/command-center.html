Add a machine

On any Mac or Linux box you want to drive from here, with tmux running, sign in once:

git clone https://github.com/cjzeroxaa/tmux-mobile && cd tmux-mobile
npm ci
node server.mjs --register https://eng.impo.ai --login

The CLI prints a Google device-login code; sign in with any allowed account. Full instructions: docs/join-network.md.

Who can see what

Open team. Anyone signed in to eng.impo.ai with an allowed Google account (sonicgg@gmail.com or any @rebyte.ai address) sees every machine on this controller, no matter who registered it.

Each agent card is tagged with the email of whoever registered it. To make a machine invisible to others, stop the agent process on that machine.
